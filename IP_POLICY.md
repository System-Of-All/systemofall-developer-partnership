# Intellectual Property (IP) Policy

The SALL Token Partnership Program requires all contributions to adhere to the following intellectual property guidelines. This policy ensures **alignment between contributor rights and the development needs of the SALL ecosystem** while protecting both parties.



## 1. Ownership
- Contributors **retain ownership** of all pre-existing intellectual property (IP) brought into the program.
- Any **new contributions** (code, research, designs, documentation, or other work product) created for the SALL ecosystem are **licensed to System Of All** for use within the ecosystem.

**Examples:**
- Code written specifically for smart contracts or infrastructure modules
- Research reports, technical papers, or analysis documents
- Design assets for platform or interface components



## 2. License
- Contributions are licensed under a **non-exclusive, royalty-free, worldwide license** for use, reproduction, modification, distribution, and display within the SALL ecosystem.
- This license **does not transfer ownership**; it allows System Of All to integrate contributions into the ecosystem while respecting contributor rights.



## 3. Attribution
- All contributions will be **credited to the contributor** in program documentation, release notes, and ecosystem acknowledgments.
- Public recognition may include:
  - Contributor listings on GitHub
  - Mentions in technical or governance documentation
  - Recognition in milestone-based program updates



## 4. Confidentiality
- All non-public contributions, communications, or materials shared within the program are **confidential** until formal approval for disclosure.
- Contributors must **not share, publish, or otherwise disclose** proprietary or sensitive materials without explicit authorization.

> By participating in the program, contributors agree to this IP policy. It balances the **rights of individual contributors** with the **needs of the ecosystem**, ensuring collaboration is productive, fair, and legally sound.
